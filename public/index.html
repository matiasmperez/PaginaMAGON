Sobre nosotros

Magon es una empresa de desarrollo de software que inicio en 2020 con dos emprendedores impulsados por la creatividad de desarrollar programas.

Los principios que orientan nuestras acciones:

  • Ejecutamos con excelencia
  • Damos lo maximo
  • Creamos valor para el usuario
  • Aprendizaje continuo

Podemos acompañarte en el desarrollo de un software empresarial, un producto a medida o una aplicación móvil; en el mejor tiempo y con la calidad requerida.

Preguntas frecuentes

  • ¿Cuantas veces puedo probar mi prueba gratuita de 30 dias?

    La prueba gratuita de 30 dias está disponible una sola vez por dispositivo.

  • ¿Si tengo mas de un dispositivo debo comprar mas de una vez el software?

    Si, el software se vende por unidad de dispositivo, esto no descarta que si tiene una cantidad grandes de dispositivos se pueda llegar a un acuerdo distinto.

  • ¿Puedo requerir una version personalizada de un software?

    Si, nuestros productos son una base para un negocio pero cada uno es personalizable por si mismo para obtener una version distinta.
